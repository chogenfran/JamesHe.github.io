C语言零基础入门1-引子

C语言入门1–引子。

一些话

很早就有这样的念头了，现在网上有的一些编程语言入门，要么废话连篇，要么在谭C的影响下只关注一些未定义行为和完全不该有的繁琐细节。

像i=i+++i+++i++这样的代码，你不论写给谁看，都会想一巴掌甩在你的脸上。

代码最重要的是简洁和可读性

而国内的很多语言入门学习类书籍完全是朝着编教材的方向去的，动辄定义 定理 例题完全不考虑学生的兴趣，教学的趣味性，误人子弟，抹杀乐趣。

再者，圈内有个怪现象就是，学的多了，连人话都不会说了，力求说出的每一句话绝对正确，顺带越晦涩难懂越好，再加上国内大部分大学老师自身教学水平有限，上课大多只是背书，简直就是在恐吓初学者。

我开始这个系列的原因就是希望能帮助对编程有兴趣而因为以上种种原因而不得其门而入的同学，当然，若是您想回顾自己所学，或帮助我查缺补漏，在下也很感谢。

那么。我们就先从准备工作开始吧。

IDE的安装

什么是IDE？
IDE就是集成开发环境（Integrated Development Environment）的英文缩写，而集成开发环境包括代码编辑器、编译器、调试器和图形用户界面工具，你可以将它理解成你编程用的工具，就像 手机是你的通讯工具一样。

程序员通过代码编辑器编辑代码，通过编译器将代码翻译成机器所能理解的机器指令，也就是可执行程序，而调试器帮助程序员调试自己的程序，找出程序漏洞和错误（BUG），图形编译器使程序员编程更方便。

会有人在你编程建议你不使用IDE，从而更了解编译原理，然而我不建议这样做，初学者应从这些细枝末节中解放出来，更关注他该学习的内容，更高效率的学习，而不是把时间浪费在配置一些东西上面。

本教程基于Visual Studio 2015
微软的Visual Studio 2015 Community是一个免费，功能强大的IDE。下载链接

下载得到的iso文件后，双击iso文件（win8/10），回到我的电脑，便出现了虚拟光驱，双击打开，或打开其中的运行文件。

选择自定义

自己选择要安装的路径，选择编程语言-Visual C++(其他的最好不要选)

然后可以看个电影，逛逛B站什么的，Visual Studio 2015安装较慢，一般取决于你的硬盘和网络速度，一般会有1个小时左右。

安装成功后，我们的准备工作就做好啦。
